# Lesson template

## Recommended section order

1. Title
2. Core idea
3. Mental model
4. What it is
5. Why it matters
6. Mechanism
7. Applied football or market example
8. Blackboard breakdown
9. Failure mode
10. Checklist
11. Practice exercise
12. Key takeaway

## Minimum expectation

If a lesson skips mechanism, failure mode, or exercise, it is incomplete.
