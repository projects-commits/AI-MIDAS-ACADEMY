### Aula 13: API REST — Consultar Mercados

```python
import requests

HOST = "https://clob.polymarket.com"
resp = requests.get(
    f"{HOST}/markets",
    params={"limit": 100, "offset": 0, "tag": "soccer", "closed": False},
)
markets = resp.json()
for m in markets["data"]:
    print(f"{m['question'][:60]:60s} | Vol: {m['volume']:>10s} | Liquidez: {m['liquidity']:>10s}")
```
