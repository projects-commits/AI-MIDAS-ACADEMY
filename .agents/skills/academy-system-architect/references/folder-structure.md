# Folder structure reference

## Public app

- `src/pages/` for routes
- `src/layouts/` for reusable page shells
- `src/components/` for reusable interface pieces
- `src/content/` for course, framework, and glossary content
- `src/data/` for supporting JSON datasets
- `src/styles/` for global tokens and UI styles

## Public assets

- `public/assets/images/`
- `public/assets/diagrams/`

## Internal tooling

- `scripts/` for validation, search indexing, and content utilities
- `private-generator/` for raw source material, prompts, and non-public outputs

## Agent system

- `.agents/skills/` for repository-scoped Codex skills
