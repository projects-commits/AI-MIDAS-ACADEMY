# Course structure reference

## Standard planning order

1. Name
2. Core promise
3. Why this course exists
4. Enemy
5. Audience
6. Prerequisites
7. Course stats
8. Tracks
9. Modules
10. Lessons
11. Labs
12. Exercises
13. Glossary terms
14. Minimum word count target

## Structural expectation

- Tracks define the broad learning lanes
- Modules define the teaching progression
- Lessons define the smallest meaningful teaching unit
- Labs convert theory into action
- Glossary terms prevent semantic drift
